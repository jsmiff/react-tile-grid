react-tile-grid
============

# About

A react tile grid that implements:

* Search
* Sorting
* Animations
* React Router

It wants:

* Forever scroll
* Unit tests like whoa

## Example

[Demo](http://iamjsmith.com/react-tile-grid/)

## Technologies

* React
* Backbone (collections)
* Browserify
* Gulp


# Usage
## Install
```bash
$ npm install
$ gem update --system
$ gem install compass
```

## Run
```bash
$ gulp
```

Visit http://localhost:8080.
